# CloudStack Pro Pricing
A fully responsive pricing section built using **HTML** and **Tailwind CSS**.

This project showcases a clean and modern pricing layout with three subscription plans:

- **Basic Plan**
- **Pro Plan (Highlighted Most Popular)**
- **Enterprise Plan**

The layout is fully responsive and adapts beautifully across:

-  Mobile devices
-  Tablets
-  Desktop screens


## Features 

### Responsive Design
The pricing cards stack vertically on smaller screens and align horizontally on medium and larger screens using Tailwind breakpoints.

### Utility-First Styling
Built entirely with Tailwind utility classes wind.

### Interactive Hover Effects
Cards animate upward on hover for a subtle premium effect.

### Highlighted Pro Plan
The Pro plan is visually emphasized using:
 - Indigo Backgroung
 - Whit Text
 - Ring border
 - blue button

 ### Mobile Optimization
Non-essential bonus features are hidden on smaller screens to keep the interface clean and uncluttered.
